# Methods Knowledge
# 方法知识库

## Purpose
## 用途

This is a library of reusable problem-solving methods. When you solve a notable problem, document the solution here.
这是可复用问题解决方法的库。当你解决一个值得注意的问题时，在这里记录解决方案。

## Directory Structure
## 目录结构

```

```

## 方法模板

## Rules
## 规则

1. Document when you solve a problem worth remembering
2. Include enough detail to recreate the solution
3. Use clear, descriptive titles
