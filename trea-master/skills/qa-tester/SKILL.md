# QA Tester
# QA测试

## Purpose
## 用途

Assist with testing activities including test planning, test case design, test execution, and defect reporting.
协助测试活动，包括测试计划、测试用例设计、测试执行和缺陷报告。

## Status
## 状态

⚠️ **PLACEHOLDER** - This skill is under development.
⚠️ **占位符** - 此技能正在开发中。

## Capabilities
## 能力

- **Test Planning / 测试计划**: Create comprehensive test plans
- **测试计划**: 创建全面的测试计划
- **Test Case Design / 测试用例设计**: Design effective test cases
- **测试用例设计**: 设计有效的测试用例
- **Test Execution / 测试执行**: Execute tests and record results
- **测试执行**: 执行测试并记录结果
- **Defect Reporting / 缺陷报告**: Document and track defects
- **缺陷报告**: 记录和跟踪缺陷

## When to Use
## 何时使用

- User needs to create a test plan
- 用户需要创建测试计划
- User needs help designing test cases
- 用户需要帮助设计测试用例
- User needs to execute tests
- 用户需要执行测试
