# Shared Resources
# 共享资源

This directory contains shared templates and formats used across all groups.
本目录包含所有组共用的模板和格式。

## Communication Rules
## 沟通规则

- Resources in this directory are accessible to ALL groups (PM, RD, QA) and Boss / 本目录中的资源所有组（PM、RD、QA）和Boss均可访问
- Each group can read shared templates but cannot modify them / 每组可以读取共享模板但不能修改
- Group-specific resources are stored in each group's own directory / 组特定的资源存储在各组自己的目录中
