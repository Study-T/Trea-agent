# Templates Knowledge
# 模板知识库

## Purpose
## 用途

This is a library of standardized templates for common tasks.
这是常见任务标准化模板的库。

## Available Templates
## 可用模板

| Template | Description | Use Case |
|----------|-------------|----------|
| meeting-notes | Meeting notes template | After meetings |
| project-plan | Project plan template | Project planning |
| code-review | Code review checklist | Code reviews |

| status-report | Status report template | Regular updates |

## Template Usage
## 模板使用

1. Find the template you need
2. Copy it to your working directory
3. Fill in the content
4. Save with appropriate naming
