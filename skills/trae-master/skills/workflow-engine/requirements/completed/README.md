# Completed Requirements
# 已完成需求

This directory contains requirement records that have completed all phases (PM→RD→QA) and passed Boss review and user confirmation.
本目录包含已完成所有阶段(PM→RD→QA)并通过Boss审核和用户确认的需求记录。

## Structure / 结构

```
completed/
├── REQ-2026-001/
│   ├── metadata.md
│   ├── prd.md
│   ├── pm-record.md
│   ├── rd-record.md
│   ├── qa-record.md
│   ├── timeline.md
│   └── summary.md
├── REQ-2026-002/
└── ...
```

## Lifecycle / 生命周期

- Requirements are moved here from `active/` after full completion
  需求在全部完成后从`active/`移至此处
- Requirement files should preserve the directory-based template layout defined in `skills/workflow-engine/requirements/RECORDER_PROTOCOL.md`
  需求文件应保持 `skills/workflow-engine/requirements/RECORDER_PROTOCOL.md` 定义的目录式模板布局
- These records serve as historical reference and knowledge base
  这些记录作为历史参考和知识库
- Do not modify completed requirement records
  不要修改已完成的需求记录
