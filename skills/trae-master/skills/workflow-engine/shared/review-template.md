# Review Template
# 审核模板

All reviews (Leader Review and Boss Review) must follow this format.
所有审核（组长审核和Boss审核）必须遵循此格式。

## Approval Format
## 通过格式

```
🔍 <reviewer_name>审核 / Review: ✅通过 / Approved
📝 评价 / Comment: <brief evaluation of the work / 对工作的简短评价>
```

## Rejection Format
## 拒绝格式

```
🔍 <reviewer_name>审核 / Review: ❌不通过 / Rejected
📦 错误信息 / Errors:
  ❌ <error 1: specific description of what is wrong / 错误1：具体描述什么有问题>
  ❌ <error 2: specific description of what is wrong / 错误2：具体描述什么有问题>
  ❌ <error 3: ...>
💡 修改建议 / Suggestions:
  - <suggestion 1: how to fix error 1 / 建议1：如何修复错误1>
  - <suggestion 2: how to fix error 2 / 建议2：如何修复错误2>
  - <suggestion 3: ...>
🔄 打回给 / Return to: <who receives the rejection / 谁接收拒绝>
```

## Review Criteria
## 审核标准

### Leader Review Criteria / 组长审核标准
- Completeness / 完整性: Did all specialists produce their required output? / 所有专员是否产出了所需的输出？
- Consistency / 一致性: Are the outputs logically consistent with each other? / 输出之间是否逻辑一致？
- Quality / 质量: Is the depth and accuracy of analysis sufficient? / 分析的深度和准确性是否充分？
- Alignment / 对齐: Does the work align with the original requirement? / 工作是否与原始需求对齐？

### Boss Review Criteria / Boss审核标准
- Direction / 方向: Is the overall direction correct? / 整体方向是否正确？
- Quality / 质量: Does the work meet expectations? / 工作是否达到期望？
- Risk / 风险: Are there any overlooked risks or gaps? / 是否有被忽视的风险或缺口？
- Feasibility / 可行性: Is the proposed solution feasible? / 提出的方案是否可行？
