# Developer
# 开发

## Purpose
## 用途

Assist with software development tasks including coding, debugging, code review, and technical implementation.
协助软件开发任务，包括编码、调试、代码审查和技术实现。

## Status
## 状态

⚠️ **PLACEHOLDER** - This skill is under development.
⚠️ **占位符** - 此技能正在开发中。

## Capabilities
## 能力

- **Coding / 编码**: Write clean, maintainable code
- **编码**: 编写清晰、可维护的代码
- **Debugging / 调试**: Find and fix issues
- **调试**: 发现并修复问题
- **Code Review / 代码审查**: Review code for quality
- **代码审查**: 审查代码质量
- **Technical Writing / 技术写作**: Document technical solutions
- **技术写作**: 记录技术方案

## When to Use
## 何时使用

- User needs to implement a feature
- 用户需要实现一个功能
- User needs help debugging an issue
- 用户需要帮助调试一个问题
- User needs code reviewed
- 用户需要代码审查

## Implementation Notes
## 实施说明

This skill should be invoked when:
此技能应在以下情况调用：

1. User asks to write or modify code
2. 用户要求编写或修改代码
3. User encounters a bug to fix
4. 用户遇到需要修复的bug
5. User needs technical documentation
6. 用户需要技术文档
