# Project Manager
# 项目经理

## Purpose
## 用途

Help with project planning, task decomposition, progress tracking, and stakeholder management.
帮助进行项目规划、任务分解、进度跟踪和干系人管理。

## Status
## 状态

⚠️ **PLACEHOLDER** - This skill is under development.
⚠️ **占位符** - 此技能正在开发中。

## Capabilities
## 能力

- **Task Decomposition / 任务分解**: Break down large tasks into manageable pieces
- **任务分解**: 将大任务分解为可管理的部分
- **Planning / 规划**: Create project plans with milestones
- **规划**: 创建带里程碑的项目计划
- **Tracking / 跟踪**: Monitor progress and identify blockers
- **跟踪**: 监控进度并识别阻碍
- **Communication / 沟通**: Draft project communications
- **沟通**: 起草项目沟通文档

## When to Use
## 何时使用

- User needs to plan a new project
- 用户需要规划一个新项目
- User needs to break down a complex task
- 用户需要分解一个复杂任务
- User needs help tracking project progress
- 用户需要帮助跟踪项目进度

## Implementation Notes
## 实施说明

This skill should be invoked when:
此技能应在以下情况调用：

1. User explicitly asks for project management help
2. 用户明确要求项目管理帮助
3. A task requires coordination of multiple people
4. 任务需要多人协调
5. User needs help breaking down a large project
6. 用户需要帮助分解大型项目
